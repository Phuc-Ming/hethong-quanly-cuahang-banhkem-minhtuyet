# Kế hoạch Sprint

## Sprint 1: Phân tích và thiết kế nền tảng

- Phân tích yêu cầu hệ thống
- Thiết kế giao diện đăng nhập
- Thiết kế giao diện trang chủ
- Thiết kế ERD

## Sprint 2: Quản lý sản phẩm

- Thiết kế giao diện sản phẩm
- Lập trình thêm, sửa, xóa sản phẩm
- Tìm kiếm và lọc sản phẩm

## Sprint 3: Đơn hàng và hóa đơn

- Thiết kế giao diện đơn hàng
- Tạo hóa đơn
- Tính tổng tiền đơn hàng
- Lưu thông tin khách hàng

## Sprint 4: Hoàn thiện hệ thống

- Quản lý kho nguyên liệu
- Quản lý khách hàng
- Quản lý nhân viên
- Thống kê doanh thu
- Kiểm thử và sửa lỗi
