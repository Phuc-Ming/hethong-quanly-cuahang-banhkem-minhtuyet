# Cơ sở dữ liệu

Thư mục này chứa các tài liệu liên quan đến cơ sở dữ liệu của hệ thống quản lý cửa hàng bánh kem Minh Tuyết.

## Nội dung

- `database.sql`: file tạo cơ sở dữ liệu và các bảng.
- `erd.png`: sơ đồ thực thể liên kết của hệ thống.

## Các bảng chính

- Sản phẩm
- Khách hàng
- Nhân viên
- Đơn hàng
- Chi tiết đơn hàng
- Nguyên liệu
