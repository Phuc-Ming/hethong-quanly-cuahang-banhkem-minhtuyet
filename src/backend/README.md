# Backend

Thư mục này dùng để lưu mã nguồn xử lý phía server trong các phiên bản phát triển tiếp theo.

Ở phiên bản demo hiện tại, hệ thống được xây dựng bằng HTML, CSS và JavaScript, dữ liệu được lưu tạm bằng localStorage để mô phỏng thao tác thêm, sửa, xóa và tra cứu dữ liệu.

Các chức năng backend dự kiến phát triển:

- API đăng nhập
- API quản lý sản phẩm
- API quản lý đơn hàng
- API quản lý hóa đơn
- API quản lý khách hàng
- API quản lý kho nguyên liệu
- API thống kê doanh thu
